# Quantum Random Number Generator (QRNG)

This project demonstrates a Quantum Random Number Generator using qubit superposition.

## Concept
A Hadamard gate is applied to a single qubit to place it in superposition. When measured, the qubit collapses randomly to 0 or 1, generating true quantum randomness.

## Gates Used
- Hadamard (H)
- Measurement

## Tool Used
IBM Quantum Composer
